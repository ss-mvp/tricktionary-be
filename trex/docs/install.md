**Requirements**

- NodeJS
- a PostGreSQL server

#

**local environment**

    PORT=8080
    DATABASE_URL=postgresql://user:pass@database:5432/
    BASE_URL=http://0.0.0.0

#

**Install**

<!-- ![install](/gifs/install.gif) -->

    git clone https://github.com/story-squad/tricktionary-be

    cd tricktionary-be

    npm install

#

**Commands**

#

    npm run build

- Clean, lint, & transpile TS -> JS.

#

    npm start

- start the app.
